data-injest

This page demonstrates the data-injest attribute natively fetching CSS / Scripts (like Tailwind and DaisyUI) dynamically and suspending DOM rendering (opacity-0) until parsing completes!

Syntax & API Usage

<div data-injest="{ css: { link: `href='...' rel='stylesheet'` }, js: { script: `src='...'` } }"></div>
