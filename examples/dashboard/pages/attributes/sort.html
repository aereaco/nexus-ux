data-sort

This page demonstrates how data-sort intercepts HTML5 Native Drag & Drop to mutate backing array signals.

Syntax & API Usage

<div data-sort="itemsArray">...</div>

Interactive Demonstration

Preview Code 
<div class="menu bg-base-200 w-full rounded-box shadow-inner min-h-64 p-4 gap-2" data-sort="kanban">

</div>

Extensibility Notes

Drag & Drop behavior in src/modules/attributes/sort.ts implements HTML5 Native Drag vectors. This can be extended to support touch interfaces by shim injecting touch-events.
