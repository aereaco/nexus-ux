data-component

This page demonstrates the data-component attribute—the revolutionary lazy-loading template injesting engine powering this entire dashboard.

Syntax & API Usage

<div data-component="/path/to/component.html"></div>

Extensibility Notes

The src/modules/attributes/component.ts loader mounts dynamically. Integrating a global HTTP cache interceptor significantly reduces template repaints in large layouts.
