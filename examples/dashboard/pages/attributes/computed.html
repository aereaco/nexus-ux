data-computed

Demonstrates the data-computed attribute dynamically injecting highly-optimized, purely derivative mathematical signals into the AST scope down-chain.

Syntax & API Usage

<div data-computed-totals="price * quantity"></div>

Extensibility Notes

The derived reactive memory inside src/modules/attributes/computed.ts maps getter traps. You can add memoization caching to this layer natively.
