data-show

This page demonstrates layout toggling modifying the native display CSS property via data-show.

Interactive Demonstration

Preview Code
I am structurally permanent but visually togglable! 
<div data-show="isVisible">Toggled Display</div>

Extensibility Notes

The visibility toggler in src/modules/attributes/show.ts strictly maps to display: none. This can be adapted to toggle CSS classes like .hidden for Tailwind synergy natively.
