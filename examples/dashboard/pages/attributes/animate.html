data-animate

Demonstrates Native WAAPI (Web Animations API) bindings mapped mathematically to HTML5 attributes using data-animate-* hooks.

Syntax & API Usage

<div data-animate="hover:scale-110"></div>

Extensibility Notes

Native element animation defined via src/modules/attributes/animate.ts natively controls the Web Animations API. Hook into this to support FLIP layout physics natively.
