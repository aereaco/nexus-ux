Extensibility Notes

Located in src/modules/attributes/signal.ts, the signal directive parses JSON object literals and creates reactive state scopes. Signals cascade from parent to child nodes via the DOM data stack, enabling true DOM-as-State composition.

  • Local signals — Scoped to the element and its descendants
  • Global signals — Use data-signal:global or data-ux-init for app-wide state
  • Nested inheritance — Child signals inherit parent scope automatically
  • Persistence — Combine with _sessionStorage, _localStorage, or _indexedDB mirrors
