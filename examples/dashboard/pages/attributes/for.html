data-for

This page demonstrates declarative list rendering mapped entirely mathematically to array state mutation via data-for.

Syntax & API Usage

<template data-for="item in items">...</template>

Extensibility Notes

Iterative rendering in src/modules/attributes/for.ts relies on deep Proxy cloning. This can be scaled physically using virtual-list mapping for 100k+ DOM node arrays.
